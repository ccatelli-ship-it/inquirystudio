Shared Values Activity

Move from individual teaching values to shared team norms that guide how teachers work together.

Stats

Suggested Time
30–45 minutes
Grouping
Teacher team
Output
3 shared values + visible team behaviors
The Arc
SurfaceClusterCommit

Shared Values helps a teacher team move from personal beliefs to collective commitments. The team surfaces individual teaching values, looks for patterns, selects three values that feel shared and practical, and translates each value into visible behaviors that can be used during planning, collaboration, and classroom implementation.

This activity is not about choosing nice words for a poster. It is about deciding how the team wants to operate together and what students should consistently experience across classrooms. By naming shared values and defining what each value looks like in action, teachers create a practical agreement they can return to when making decisions, designing learning experiences, addressing tension, and strengthening belonging. The strongest shared values are relevant, observable, and realistic enough to shape day-to-day practice.

Steps

01

Post individual teaching values. Each team member posts their final 3–5 teaching values on a shared board, chart paper, or digital space.

02

Add classroom evidence. As each person posts, they add one quick example of how each value currently shows up in their classroom. Examples can include routines, language, student roles, relationships, assignments, feedback, or norms.

03

Listen for patterns. Team members silently review the board. Notice repeated words, similar ideas, and values that seem to point toward the same kind of student experience.

04

Cluster similar values. Group related values together. Give each cluster a short theme name that captures what seems to matter most to the team.

05

Select three shared values. Use a quick vote, dot vote, or consensus check to select three values that feel shared, relevant, and practical to demonstrate across classrooms.

06

Define each value in action. For each shared value, complete this sentence: "Students will know we value ___ when they consistently experience ___." Push for behaviors that are visible and specific.

07

Name common team moves. Identify 1–2 moves the team can reliably use across classrooms to make each value visible. Focus on practices the team can actually repeat.

08

Use the values as a design filter. Before finalizing a project, lesson, or team decision, ask: "Which shared value does this strengthen? Which value needs to be more visible?" Revise the design so the values are felt by students.

Value-in-Action Frame

Students will know we value [shared value] when they consistently experience [visible behavior].
